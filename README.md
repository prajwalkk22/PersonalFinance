Problem Statement

Most people:
Don’t know where their money goes
Miss potential tax deductions
Struggle with budgeting consistency
Lack actionable financial insights

Our solution provides real-time tax-aware financial insights with an intuitive, secure, and user-friendly experience.


Target 1 – Setup Authentication
Create user registration and login functionality, securely hash passwords, generate JWT tokens, and protect private routes using authentication middleware.

Target 2 – Add Transaction Management
Implement APIs to add income and expense transactions, fetch user-specific transactions, and delete transactions securely.

Target 3 – Implement Tax Calculation Logic
Build a tax engine that calculates estimated deductions for eligible expenses and maintains a running tax impact summary for the user.

 Target 4 – Build Dashboard Analytics
Create an endpoint that calculates total income, total expenses, savings, tax estimates, and category-wise spending breakdown.

Target 5 – Add Budget Management
Allow users to create monthly category budgets and compare actual spending against budget limits with overspending detection.

Target 6 – Implement Goal-Based Savings
Enable users to create savings goals, track current progress, and update progress automatically based on savings.

Target 7 – Add Financial Health Score
Calculate a financial score based on savings rate, budget adherence, and tax optimization, and return it via an API.

Target 8 – Secure the Backend
Add input validation, error handling, rate limiting, and ensure users can only access their own financial data.
